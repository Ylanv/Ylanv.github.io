Byte-Pair Encoding (BPE)

A tokenization algorithm used in NLP.

Explanation

Byte-Pair Encoding (BPE) is a simple and efficient way to tokenize text by replacing frequently occurring pairs of symbols with a single symbol.

Algorithm

def byte_pair_encoding(text, vocab_size):
    # Initialize with all characters
    vocab = {char: text.count(char) for char in set(text)}
    while len(vocab) < vocab_size:
        # Find most common pair
        pairs = {}
        for word in text.split():
            for i in range(len(word) - 1):
                pair = word[i:i + 2]
                pairs[pair] = pairs.get(pair, 0) + 1
        most_common = max(pairs, key=pairs.get)
        text = text.replace(most_common, ''.join(most_common))
    return text
